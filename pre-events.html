Code Debugging

                Code debugging is a competition for all those coding enthusiast's Each individual contestant will be given a code with some errors (mainly logical errors) . Points will be awarded based on errors cleared. There will be two stages for this competition.

Prizes worth 3k awaits you...

Date : 20/03/19
Time : 10:00 AM

Registration Fee: ₹200/- per head

RULES

1. Participants must register their names in online on or before.
2. This code debugging contest will be held offline.
3. During contest, no participant is allowed to browse the internet. If any case that participant will be eliminated from the list.
4. Students should participate as a individual member.
5. Programming language used is C.
6. Participants are not allowed to discuss any aspects of the problem with others during challenge.
7. All decision made by the judges are final and are not subject to appeal.
8. Event in-charge will assign code to Debug.
9. All teams will be given the same set of problems to solve.
10. Each problem will have a specified point value. The more difficult the problem, the more points a correct solution will receive.
11. After debugging, each program will be tested based on our critical test data. However output and input should be exactly as specified in the samples provided.
12. Winners are determined strictly based on overall score as determined by the scoring.
